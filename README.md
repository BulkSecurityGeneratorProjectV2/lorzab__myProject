# Lora Bahr Individual Project

###Problem Statement
I enjoy reading and do many of my family however I tend to lend my books or sell them back to buy more.  I want to recommend what I have read to my family but forget the titles at times.  If I had a list of the books that I have read and a place to put any note about what I think others should know and let them see it would be helpful.  I would like to be able to see what they have read too and if I do not know what I want to read next have a recommendation page to choose from.
